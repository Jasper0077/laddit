# About

Laddit, read as "la - dit", is a demo project a fullstack Node.js based web application.

## Prerequisites

In this project, we used the following technologies:

* Typescript
* React.js
* Node.js
* Next.js
* Apollo
* Express.js
* Redis
* PostgreQL
* GraphQL
* Chakra-ui
* Tailwind
